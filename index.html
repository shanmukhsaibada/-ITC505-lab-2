Addendum: Explanation of HTML Tags

  • <html>: The root element that defines the beginning and end of the HTML document.
  • <head>: Contains metadata and the title of the document.
  • <title>: Sets the title of the document, which appears in the browser tab.
  • <body>: Contains the content of the HTML document that is visible to users.
  • <div class="container">: A container for grouping and styling content.
  • <header>: Defines the introductory content or navigation links for the document.
  • <h1>: Represents the main heading of the document or section.
  • <p>: Defines a paragraph of text.
  • <a>: Defines a hyperlink to another webpage or resource.
  • <section>: Groups related content into sections.
  • <h2>: Represents a subheading within the document or section.
  • <div class="education">: A container specifically for grouping educational details.
  • <h3>: Represents a heading for subsections within a section.
  • <ul>: Defines an unordered list of items.
  • <li>: Defines a list item within an ordered or unordered list.
  • <div class="project">: A container specifically for grouping project details.
